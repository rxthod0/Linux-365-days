Day 4 Practice - Linux 365 Days

Topics Covered:
- find : locate files in directories
- grep : search for words inside files
- wildcards (*, ?) : match file patterns
- wc : count lines, words, characters
- sort : arrange file contents alphabetically
- uniq : remove duplicate lines
- pipes (|) : combine commands together

Files Created:
- notes.txt : summary of topics
- logfile.txt : sample log file for grep/wc
- names.txt : sample names for sort/uniq
- misc.txt : extra files for wildcard practice
- commands.txt : commands used and outputs
- summary.txt : final notes

Goal:
To master Linux text processing and search commands by solving practical challenges.
